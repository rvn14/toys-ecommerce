# Global Toys Store

A full-stack global toys and collectibles e-commerce platform built with Go, Gin, GORM, PostgreSQL, Next.js, TypeScript, and Tailwind CSS.

## Project Theme

Global Toys Store is an e-commerce web application for toys, collectibles, building sets, action figures, comic merchandise, gaming merchandise, board games, plush toys, and safe toy props.

## Tech Stack

### Backend
- Go
- Gin
- GORM
- PostgreSQL
- JWT authentication
- bcrypt password hashing
- REST API
- Clean/layered architecture

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Database
- PostgreSQL
